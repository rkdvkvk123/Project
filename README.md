* * *

  + Apache tomcat 9.0   
+ Dev tool
   + Eclipse
   + Sql developer   
+ Front-End
   + HTML5
   + CSS
   + JavaScript
   + JQuery
   + Ajax
+ Back-End
   + JSP
   + Spring
   + MyBatis   
* * *   
## 개발기간   
+ 개발기간 : 2020.08~ 2020.10  
+ 개발인원 : 6명

## 기능   
> 1.User)로그인,회원가입   
> 2.User)해당기업지원,리뷰작성   
> 3.Admin)채용공고리스드 등록,수정,삭제   

## 개발배경
> Java,Jsp,Spring,MyBatis 를 이용하여 DataBase의 Data를 운용 및 활용할 수 있다.   

## 개발 목적
> 웹과 Maven MVC 방식을 숙지한다.
